A basic foundation of design is Typography − This mixin is aimed to create typographic rhythm with css styles using Sass. It calculates the appropriate font-size, line-height, margin-bottom and padding in Ems while keeping it in a neat 24px Base Vertical Grid. 

pre {
  @include typoRhythm(21, $margin: 1, $padding: 1);
}

pre {
  font-size: 1.3125em;
  padding: 2.28571em;
  margin-bottom: 1.14286em;
  line-height: 1.14286em;
}
